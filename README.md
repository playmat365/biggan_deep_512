# biggan-deep-512 图片素材

* 使用 tensorflow 2.3 运行
* 共 1000 类，每类生成 100 张图片，以 jpg 格式储存

网页版：[https://playmat365.github.io/biggan_deep_512/](https://playmat365.github.io/biggan_deep_512/)
